# Hospital Appointment Booking System

## Overview
This is a Java-based console application that helps a hospital or clinic manage patient details, doctor information, and appointment bookings. It allows users to register patients, manage doctors, book or cancel appointments, and view appointment schedules.

## Features
- Patient registration and listing
- Doctor registration and listing with specialization
- Search doctors by specialization
- Book appointments between patient and doctor
- Prevent double-booking of appointments
- View appointments by patient or by date
- Cancel existing appointments

## Technologies Used
- Java (Core Java, OOP concepts)
- Collections Framework (ArrayList, etc.)
- UML for design (Use Case, Class, Sequence, Workflow)
- (Optional) File handling for data persistence
